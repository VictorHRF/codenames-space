# Contexto del Proyecto: Codenames Space

Descripción: Este es un juego web multijugador basado en deducción social inspirado en Codenames, pero con una temática espacial de rescate.

Historia: Una tormenta magnética dispersó las flotas de dos razas alienígenas: Los Zirtox (verdes) y los Nebulis (rosas). Los Jefes de Flota desde la Estación Espacial dan pistas encriptadas a la Nave Nodriza para rescatar sus naves, evitando a las naves humanas (transeúntes) y al Agujero Negro (muerte súbita).

Stack Tecnológico:
* Frontend & Backend: React y Next.js (App Router).
* Estilos: Tailwind CSS (diseño Sci-Fi, Dark Mode, Glassmorphism).
* Base de Datos y Tiempo Real: Supabase.

Reglas para la IA:

Al generar código, asegúrate de usar componentes de cliente ('use client') solo cuando interactúen con el estado o hooks, y Server Actions para la lógica segura.
* Los colores principales son verde neón para los Zirtox y rosa para los Nebulis.
